# WPS to PDF

Convert Microsoft Works word processor files into PDF. Microsoft Works was bundled with consumer PCs throughout the late 1990s and 2000s, and plenty of WPS files still live on old hard drives and email archives. This tool opens them in the browser and produces clean PDFs.

## Supported Formats

| Format          | Extension | Notes                                     |
| :-------------- | :-------- | :---------------------------------------- |
| Microsoft Works | `.wps`    | Word processor files from Microsoft Works |

## How It Works

1. Click the upload area or drag and drop your WPS files. Multiple files are accepted.
2. Review the file list. Use the trash icon to remove individual files, or click **Add More Files** to expand your batch.
3. Click **Convert to PDF**. A LibreOffice-based engine loads in the browser and converts each WPS document to PDF.
4. One file downloads as a PDF directly. Multiple files are bundled into a ZIP archive named `wps-to-pdf.zip`.

## Options

No additional settings are needed. The converter handles text, basic formatting, and embedded images from the WPS file.

## Use Cases

- Recovering old school or work documents saved in Microsoft Works format
- Opening WPS email attachments when you don't have Microsoft Works installed
- Converting an archive of WPS files for migration to a modern document system
- Printing legacy documents that current word processors cannot open

## Tips

- Do not confuse Microsoft Works `.wps` files with WPS Office files. This tool is specifically for the older Microsoft Works format.
- The LibreOffice engine takes a moment to load on first use. Once loaded, additional conversions in the same session run faster.
