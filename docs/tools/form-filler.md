# PDF Form Filler

Open a PDF with form fields and fill it out directly in an embedded viewer. The tool supports both standard AcroForm fields and legacy XFA (XML Forms Architecture) forms. Once you have completed the form, download the filled version with a single click.

## How It Works

1. Upload a PDF that contains form fields.
2. The document opens in an integrated PDF viewer that renders form fields as interactive inputs.
3. Click on text fields, checkboxes, dropdowns, and radio buttons to fill them in.
4. Click **Save & Download Filled Form** to get the completed PDF.

## Features

- Embedded PDF.js viewer with full form field interaction
- Support for standard AcroForm fields (text inputs, checkboxes, dropdowns, radio buttons)
- XFA form support for legacy XML-based forms
- Automatic viewer sizing based on page dimensions
- File display with name and size information

## Options

There are no additional configuration options. Upload the form, fill it in, and download.

## Use Cases

- Completing tax forms, insurance applications, or government paperwork
- Filling out job application forms received as PDF attachments
- Completing vendor onboarding forms or supplier questionnaires
- Signing off on compliance checklists with checkbox fields
- Filling medical intake forms before an appointment

## Tips

- XFA forms are a legacy format only supported by some PDF viewers. If you open an XFA PDF in Chrome or most other tools and see blank pages, it means those viewers lack XFA support. BentoPDF's Form Filler and Firefox both handle XFA properly.
- For PDFs without existing form fields, use the [Create PDF Form](./form-creator) tool to add fields first, then fill them here.
- The viewer automatically adjusts its size to match the PDF's page dimensions.
