# VSD to PDF

Convert Microsoft Visio diagrams into PDF without installing Visio. VSD and VSDX files contain flowcharts, org charts, network diagrams, and floor plans that most people cannot open. This tool renders them as clean PDFs viewable on any device.

## Supported Formats

| Format       | Extension | Notes                                        |
| :----------- | :-------- | :------------------------------------------- |
| Visio Binary | `.vsd`    | Legacy Visio format (Visio 2003 and earlier) |
| Visio XML    | `.vsdx`   | Modern Visio format (Visio 2013+)            |

## How It Works

1. Click the upload area or drag and drop your VSD or VSDX files. Both formats can be mixed in a single batch.
2. Review the file list. Remove unwanted entries with the trash icon, or use **Add More Files** to add more diagrams.
3. Click **Convert to PDF**. The LibreOffice-based engine loads in the browser and renders each diagram into a PDF, preserving shapes, connectors, labels, and page layout.
4. A single diagram downloads directly as a PDF. Multiple files produce a ZIP archive named `vsd-to-pdf.zip`.

## Options

No configuration is required. The converter respects the original page size and orientation from the Visio document.

## Use Cases

- Reviewing Visio diagrams from a colleague when you don't have a Visio license
- Including network or architecture diagrams in project documentation as PDFs
- Printing org charts or process flows for meeting handouts
- Archiving legacy VSD files from older Visio versions in a future-proof format

## Tips

- Multi-page Visio documents produce multi-page PDFs, with each Visio page mapping to a PDF page.
- VSDX files (the modern XML-based format) generally convert with higher fidelity than older VSD binary files.
- The LibreOffice engine loads once per session. Expect a brief wait on the first conversion, with faster results on subsequent files.
