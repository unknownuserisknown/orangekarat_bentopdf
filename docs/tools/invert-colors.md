# Invert Colors

Flip every color in your PDF to its inverse. White backgrounds become black, black text becomes white, and all other colors shift to their complementary values. This is the fastest way to create a "dark mode" version of any PDF.

## How It Works

1. Upload your PDF file.
2. Click **Invert Colors** to process.
3. The inverted PDF downloads automatically.

There are no additional settings. The tool renders each page as an image, inverts all pixel colors, and reassembles the result into a new PDF.

## Features

- One-click color inversion for all pages
- Processes every page in the document
- Maintains original page dimensions and layout
- Works with any PDF regardless of content type (text, images, mixed)

## Use Cases

- Creating dark-mode versions of textbooks or manuals for comfortable nighttime reading
- Preparing high-contrast versions of documents for accessibility purposes
- Inverting scanned documents that were captured with incorrect exposure
- Generating negative versions of image-heavy PDFs for visual analysis

## Tips

- Since the tool rasterizes pages to apply the effect, the output is image-based. Text will no longer be selectable or searchable. If you need searchable text after inversion, run the output through an OCR tool.
- For more granular color adjustments (brightness, contrast, saturation), use the [Adjust Colors](./adjust-colors) tool instead.
- Large PDFs with many pages may take longer to process since each page is individually rendered and inverted.
