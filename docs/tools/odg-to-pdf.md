# ODG to PDF

Convert OpenDocument Graphics files into PDF. ODG is the native format for LibreOffice Draw and is used for diagrams, flowcharts, and vector illustrations. Converting to PDF makes these drawings easy to share, print, and embed in other documents.

## Supported Formats

| Format                | Extension | Notes                                    |
| :-------------------- | :-------- | :--------------------------------------- |
| OpenDocument Graphics | `.odg`    | LibreOffice Draw / OpenOffice Draw files |

## How It Works

1. Click the upload area or drag and drop your ODG files. Multiple files can be added at once.
2. Review the file list. Remove individual entries with the trash icon, or click **Add More Files** to add more drawings.
3. Click **Convert to PDF**. A LibreOffice-based engine loads in the browser and converts each ODG file, preserving vector graphics, shapes, and text.
4. One file downloads as a PDF directly. Multiple files produce a ZIP archive named `odg-to-pdf.zip`.

## Options

No additional settings are required. The conversion preserves the original page dimensions, shapes, connectors, and text boxes from the ODG source.

## Use Cases

- Sharing LibreOffice Draw diagrams with clients who don't have LibreOffice installed
- Printing flowcharts and network diagrams created in Draw
- Including ODG illustrations in a report or presentation as embedded PDFs
- Archiving technical drawings in a format that will remain readable for decades

## Tips

- Vector elements stay as vectors in the PDF output, so the result scales cleanly to any print size.
- The LibreOffice engine takes a moment to initialize on first use. Conversions within the same session after that are faster.
